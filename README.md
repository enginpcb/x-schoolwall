# x-schoolwall 校园社交系统
一款专为校园场景打造的轻量级社交平台，支持用户注册、发帖、互动、私信、个人主页等完整校园社交功能，采用 PHP + MySQL 开发，易于部署、二次开发与扩展。

## 📁 项目目录结构详解

| 路径 | 类型 | 详细说明 |
| :--- | :--- | :--- |
| `/` | 根目录 | 项目主目录，包含入口文件、服务器配置、安装脚本、说明文档 |
| `/index.php` | 入口文件 | 网站前台主入口，处理页面渲染、路由分发 |
| `/api.php` | 接口入口 | 统一 API 请求入口，用于 APP、小程序、前端 Ajax 交互 |
| `/update.php` | 升级文件 | 版本更新、数据库结构升级、数据迁移脚本 |
| `/install.sql` | 数据库文件 | 项目初始化数据库结构，包含用户、帖子、评论、消息等表 |
| `.htaccess` | 配置文件 | Apache 环境伪静态、防盗链、权限控制规则 |
| `/nginx.conf` | 配置文件 | Nginx 环境站点配置示例 |
| `/web.config` | 配置文件 | Windows IIS 服务器环境配置 |
| `/安装教程.txt` | 部署文档 | 详细手动部署步骤 |
| `/搭建教程.txt` | 部署文档 | 快速搭建教程 |
| `/BingSiteAuth.xml` | 验证文件 | 必应搜索引擎站长平台验证文件 |
| `/core/` | 核心框架目录 | 系统核心逻辑、全局配置、公共函数、初始化流程 |
| `/core/apps/` | 业务模块 | 各功能模块：登录、注册、发帖、评论、私信、举报等 |
| `/core/components/` | 通用组件 | 数据库操作、缓存、上传、加密、请求验证、邮件发送等工具类 |
| `/core/langs/` | 语言包 | 多语言翻译配置，支持中英文切换 |
| `/core/settings.php` | 系统配置 | 站点全局配置文件：数据库信息、站点名称、开关控制等 |
| `/core/definitions.php` | 常量定义 | 路径常量、状态码、业务常量统一定义 |
| `/core/web_req_init.php` | 请求初始化 | 前台页面请求初始化：会话、鉴权、全局变量加载 |
| `/core/api_req_init.php` | API 初始化 | API 接口请求初始化：鉴权、跨域、参数过滤 |
| `/themes/` | 前端主题 | 所有前端模板、UI 结构存放目录 |
| `/themes/default/` | 默认主题 | 系统默认前台主题，包含注册、登录、首页、个人中心等模板 |
| `/css/` | 样式目录 | 全局公共 CSS 样式文件 |
| `/js/` | 脚本目录 | 全局公共 JavaScript 交互逻辑 |
| `/images/` | 静态图片 | 系统默认图标、背景、占位图等 |
| `/upload/` | 用户上传 | 所有用户上传文件存储目录 |
| `/upload/avatars/` | 头像目录 | 用户头像存储 |
| `/upload/covers/` | 封面目录 | 用户主页封面、帖子封面 |
| `/upload/images/` | 帖子图片 | 发帖上传的图片资源 |
| `/install/` | 安装程序 | 可视化安装向导，自动检测环境、写入配置、导入数据库 |
| `/伪静态/` | 规则目录 | 存放 Apache / Nginx 伪静态规则文本 |
| `/api_docs/` | 接口文档 | API 接口说明、调用示例、权限说明文档 |
| `/site_backups/` | 备份目录 | 系统自动备份文件：数据库、配置、静态资源 |
| `/temps/` | 临时目录 | 缓存文件、会话文件、临时上传文件 |
| `/sitemap/` | 站点地图 | SEO 站点地图，用于搜索引擎收录优化 |
| `/apps/` | 扩展模块 | 原生 APP 对接、第三方扩展功能模块 |

## 🧩 核心功能

- 用户注册、登录、找回密码
- 个人主页、头像/封面上传
- 发布帖子、图文混排
- 评论、点赞、分享
- 私信聊天、好友系统
- 后台管理（用户/内容/权限）
- 全站搜索、内容筛选
- 多终端适配（PC + 移动端）
- 数据备份、系统升级
- API 接口支持二次开发

## 🖥 运行环境要求

- PHP >= 7.2
- MySQL >= 5.6 / MariaDB
- Apache / Nginx / IIS
- 开启 GD 库、fileinfo、mbstring、curl 扩展

## 🚀 快速部署

1. 上传源码至网站根目录
2. 访问 `域名/install` 进入安装向导
3. 填写数据库信息、管理员账号
4. 安装完成后删除 `/install` 目录
5. 配置伪静态规则（详见 `/伪静态` 目录）
6. 登录后台开始使用
